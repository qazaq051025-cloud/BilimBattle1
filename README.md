# ⚡ BilimBattle — Оқу Шайқасы

Мұғалімдерге арналған геймификация платформасы. Кез-келген пәнге жарайды.

## 🚀 Жылдам бастау

1. `index.html` файлын браузерде ашыңыз (VS Code Live Server, немесе тікелей екі рет шертіңіз)
2. Немесе барлық файлдарды веб-хостингке жүктеңіз

## 🔐 Кіру деректері

| Рөл | Логин | Пароль |
|-----|-------|--------|
| Admin | `ERS` | `0525` |
| Мұғалім | Өзі тіркеледі | — |

## 📁 Файл құрылымы

```
BilimBattle/
├── index.html          — Негізгі HTML
├── css/
│   ├── main.css        — Негізгі стильдер
│   └── animations.css  — Анимациялар
└── js/
    ├── data.js         — localStorage + деректер
    ├── auth.js         — Тіркелу / Кіру / Admin
    ├── editor.js       — Тест редакторы
    ├── ai.js           — AI сұрақ генераторы
    ├── game.js         — Ойын логикасы + BilimBattle режимі
    ├── monster.js      — Monster Raid режимі
    ├── ui.js           — Экран навигациясы + UI
    └── main.js         — Бастапқы нүкте
```

## 🎮 Ойын режимдері

- 🃏 **BilimBattle** — Командалық карточка ойыны (Bamboozle стилі)
- 👾 **Monster Raid** — Барлық топ Boss монстрға қарсы шайқасады

## ✏️ Тест жасау

### Қолмен
1. Редакторда «+ Сұрақ қосу» батырмасын басыңыз
2. Сұрақ мәтіні мен 4 жауапты толтырыңыз
3. Дұрыс жауаптың әріпіне басыңыз (жасыл болады)
4. «💾 Сақтау» батырмасын басыңыз

### AI арқылы
1. Тақырыпты жазыңыз (мысалы: «Квадрат түбір, 8-сынып»)
2. Сұрақ санын таңдаңыз
3. «⚡ Генерация» батырмасын басыңыз

### Файлдан импорт (.txt)
Файл форматы:
```
Сұрақ мәтіні?
A) Бірінші жауап
B) Екінші жауап
C) Үшінші жауап
D) Төртінші жауап
Дұрыс: A
```

## 📱 Технологиялар
- Таза HTML5 + CSS3 + Vanilla JavaScript
- localStorage (серверсіз)
- Anthropic API (AI генератор үшін)

## 💡 Ескертпе
AI генератор жұмыс істеуі үшін Anthropic API кілті қажет.
Сайтты Anthropic платформасы арқылы немесе API проксиімен іске қосыңыз.
